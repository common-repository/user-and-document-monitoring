== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload Plugin folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check for "Client Login" in the wordpress menu, and go to sub menu, "Settings". <br>
More information at http://hiteshjoshi.com/wordpress/client_document_monitoring.html

== Frequently Asked Questions ==

= When I load the user side page, nothing happens. Why? =

Please check if any jQuery plugin is loading in footer. This plugin is going to user MooTools after a short revision , then there will be no issue, but for now. Please don't load any jQuery plugin at footer.

= Can I assign One File to Multiple Users =

No, for this moment you can't do that. But you can assign multiple files to one user.
